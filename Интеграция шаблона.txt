ИНТЕГРАЦИЯ ШАБЛОНА.
--------------------------

Копируем папку с версткой в корень сайта
 Проверяем верстку (открываем в браузере)

Расположение: /local/ ??

Админка
  Настройки
  Настройки продукта
  Сайты
  Шаблоны сайтов
    создать шаблон, вставить #WORK_AREA#
(появится /bitrix/templates/<ID_шаблона>)
      сделать резервн. копию шаблона
  перенести папки из нов.шаблона в /bitrix/templates/<ID_шаблона>
  перенести header.php (плюс хлебн.крошки и заголовок стр.), footer.php
(или вырезать из нов.шаблона)
  Контекстную часть в корень сайта index.php
  Прописать в путях: <?=SITE_TEMPLATE_PATH?>/

<?php $bIsMainPage = $APPLICATION->GetCurPage(false) == SITE_DIR;?> (текущ.стр. - главная?)
<?php $APPLICATION->ShowHead();?> (вместо мета-тегов)
<?php $APPLICATION->ShowTitle();?>
<?php $APPLICATION->SetAdditionalCSS(SITE_TEMPLATE_PATH.'/css/common-style.css');?>

для IE:
<script src="<?=CUtil::GetAdditionalFileURL((SITE_TEMPLATE_PATH.'/js/ie.js')?>"></script>

В настройках сайта включить шаблон.

СТРУКТУРА /bitrix/templates/<ID_шаблона>:
style.css           - стили визуальн. редак-ра
template_styles.css - стили шаблона

ПОДКЛЮЧИТЬ МЕНЮ
https://www.youtube.com/watch?v=PkOEo1OHgQI&list=PLY4rE9dstrJyqAWUVlT7PLGjdmPEwhfg6&index=3

Раздела 1-го уровня меню созд-ся только с главной стр.
 После созд-я раздела система автомит-ки переадрис-ет на созданный раздел.
  Из него мы переходим на гланую стр. и создаем след-й раздел.

СОЗДАТЬ НОВЫЙ ТИП МЕНЮ (если нужен):
Админ
  Настройки
    Настройки продукта
      Управление сайтом
        Типы меню

СОЗДАТЬ ГЛАВНОЕ МЕНЮ:
Главная стр.
  Создать раздел (не радактир. стран-у),
    созд. все разделы пустыми,
  вывести само меню на главн.стр.:
    Изменить стран-у,
      убрать лишнее, добавить комп-т Меню,
      копир. код из визуальн. редактора (вызов комп-та),
        вставляем в шаблон сайта
    Отменить изменения в визуальном ред-ре.

Обновит стр.
  копир.шаблон меню из режима правки
(появится /bitrix/templates/<ID_шаблона>/components/bitrix/menu/<ID_меню>)
  редактир. файлы в /bitrix/templates/<ID_шаблона>/components/bitrix/menu/<ID_меню>
(сюда вставляем ВЕРСТКУ меню)
  удаляем стили в styles.php (сам файл оставляем)
  из режима правки добавляем ссылку на главн.стр.

    "Стр.раздела" (1-ю часть кода до слэша)
      Сохранить
