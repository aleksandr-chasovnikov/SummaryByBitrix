


template.php (в компонентах) не используется при использовании кеша.

Управление автокешированием:
	Настройки
	 Настройки продукта
	  Автокеширование

---------------------------
СБРОСИТЬ КЕШ:

1) Сайт > Сбросить кеш
2) В режиме Правки сайта использовать кнопки для очистки кеша в панели отдельных компонентов.
3) Использовать автоматический сброс кеша по истечении времени кеширования, для чего в настройках компонента выбрать режим кеширования Кешировать или Авто + Управляемое.
4) Использовать автоматический сброс кеша при изменении данных, для чего в настройках компонента выбрать режим кеширования Авто + Управляемое.
5) Перейти к настройкам выбранных компонентов и перевести их в режим работы без кеширования.

--------------------------
ТЕГИРОВАНSQ КЕШ (УПРАВЛЯЕМЫЙ):

Сброс кеша: CIBlock::clearIblockTagCache( 7 );

Для отключения тегирования конкретных инфоблоков:
CIBlock::DisableTagCache($iblock_id)

Для запуска механизма необходимо определить константу в файле dbconn.php. 
define("BX_COMP_MANAGED_CACHE", true);
(Это можно сделать простым включением Управляемого кеша на закладке Управляемый кеш на странице Настройки > Настройки продукта > Автокеширование.)

--------------------------
Управляемый кеш инфоблоков очищается только при вызове Update.
При изменении, например, свойств с помощью SetPropertyValueCode очистки не произойдет. 
Делаем вручную после изменения:
if (defined('BX_COMP_MANAGED_CACHE')) {
   $GLOBALS['CACHE_MANAGER']->ClearByTag('iblock_id_'.$arParams['IBLOCK_ID']);
}

--------------------------
